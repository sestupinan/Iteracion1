Esta aplicación fue basada y modificada desde Parranderos-JDO 
del curso Sistemas Transaccionales de la Universidad de los Andes
Para instalar la aplicación:
1. Descomprimir la carpeta
2. Se importa el proyecto st-EPSAndes en Java.
3. Se corre la app en la clase MVC como una JavaAplication.
4. La interfaz se usa en la consola del IDE.
